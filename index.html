All City

AllCity is a working, interactive, mobile application that allows users to learn more about graffiti art. This mobile application allows users to check out graffiti and street art from around the globe, post their own pictures, as well as like and comment on their favorite pieces. The API was built using Django and deployed to Heroku. The client was built using ReactNative and is in testing on the Apple App Store.
